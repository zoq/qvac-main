# QVAC OCR Addon v0.1.4 Release Notes

This release fixes a workflow permissions issue that was causing CI failures in the reusable prebuilds workflow.

## Bug Fixes

### Workflow Permissions Fix

Fixed a validation error in the reusable prebuilds workflow. The `prebuild` job in `prebuilds.yaml` incorrectly declared `pull-requests: write` permission, but none of its steps actually required this permission. When the workflow was called from `on-merge.yaml`, this caused a workflow validation error. The unnecessary permission has been removed, allowing the prebuilds workflow to run correctly.
