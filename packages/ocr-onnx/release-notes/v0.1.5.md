# QVAC OCR Addon v0.1.5 Release Notes

This release adds a supported platforms section to the README and updates the Linux ARM64 build to use Ubuntu 22.04 for consistent glibc compatibility.

## Documentation

### Supported Platforms

The README now includes a supported platforms table listing all Tier 1 platforms with their supported architectures and minimum OS versions: macOS (arm64/x64, 14.0+), iOS (arm64, 17.0+), Linux (arm64/x64, Ubuntu 22+), Android (arm64, 12+), and Windows (x64, 10+).

## Infrastructure

### Linux ARM64 Build Compatibility

Linux ARM64 builds now use Ubuntu 22.04 to match x64 builds, ensuring consistent glibc compatibility across architectures.
