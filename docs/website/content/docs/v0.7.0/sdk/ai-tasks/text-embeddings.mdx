## Overview

Text embeddings uses [`llama.cpp`](https://github.com/ggml-org/llama.cpp) as inference engine. Load any supported model using `modelType: "embeddings"`. Then, provide text input as `text` where the value is either a single `string` or an array of strings.

`embed()` returns a single embedding vector (`number[]`) for single text input, or an array of embedding vectors (`number[][]`) for batch input.

## Functions

Use the following sequence of function calls:
1. [`loadModel()`](/v0.7.0/sdk/api/loadModel)
2. [`embed()`](/v0.7.0/sdk/api/embed)
3. [`unloadModel()`](/v0.7.0/sdk/api/unloadModel)

For how to use each function, see [SDK — API reference](/v0.7.0/sdk/api/).

## Models

You can load any `llama.cpp`-compatible embeddings model. Model file format: `*.gguf`.

- If the model is sharded across multiple files (a multi-file bundle), see [Sharded models](/v0.7.0/sdk/utilities/sharded-models).
- For models available as constants, see [SDK — Models](/v0.7.0/sdk#models).
