```typescript
function ragChunk(params: Object): Promise
```

## Description

Chunks documents into smaller pieces for embedding.
Part of the segregated flow: ragChunk() → embed() → ragSaveEmbeddings()

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| `params` | `Object` | ✓ | The parameters for chunking |

## Returns

```typescript
Promise
```



## Examples

```typescript
```typescript
const chunks = await ragChunk({
  documents: ["Long document text here..."],
  chunkOpts: {
    chunkSize: 256,
    chunkOverlap: 50,
    chunkStrategy: "paragraph",
  },
});
```
